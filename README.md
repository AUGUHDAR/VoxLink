VoxLink —— 和你的好友快速进行联机，或寻找志同道合的玩伴 :D
 
开启存档，点击创建房间即可生成6位房间码。将房间码分享给好友，对方输入后等待连接成功就能加入你的世界。无需配置端口映射、修改路由器、自建服务，也不需要了解公网IP相关知识。安装模组，创建房间，分享房间码，一键搞定联机。
 
当前为Beta测试版本，联机稳定性有限 T_T。部分场景秒连，部分网络环境下会持续连接失败，联机成功率高度依赖双方网络：
 
- 高成功率：同运营商家用宽带
- 兼容性较差：校园网、企业内网、手机热点
- 极难连通：任意一方为对称NAT环境
 
连接失败时，双方重启游戏刷新NAT端口后重试即可。我们会持续迭代优化打洞逻辑，兼容更多类型的NAT网络 >_<
 
关于房间
 
房间无人数上限，支持非正版账号联机；若将房间设为大厅公开，房间名称必须经过AI审核才会对外展示。若AI服务离线/审核异常，大厅内统一展示默认房间名。
 
房间大厅
 
创建房间时可选择公开，房间会展示在公共大厅，其他玩家可一键加入。大厅不会暴露你的真实IP地址，所有联机流量均通过P2P打洞直连。房间附带分类标签，方便快速匹配玩伴。
 
离线模式
 
两名非正版玩家也可正常联机。创建房间时模组自动切换离线鉴权，访客进入后自动恢复原有配置，全程无需手动调整任何设置。
 
Bug反馈方式
 
前往GitHub提交Issues，务必上传房主与访客双方的完整日志：
日志路径： .minecraft/versions/游戏版本/logs/latest.log 
同时说明双方网络类型（家用宽带/校园网/手机热点等），缺少日志很难定位问题 orz
 
技术实现
 
采用UDP打洞 + STUN协议识别NAT类型，打洞逻辑参考EasyTier。连接建立后使用自研可靠UDP传输承载游戏流量。
信令服务器仅在握手阶段交换双方网络地址，游戏实际数据全程P2P直连，不经过中转服务器。
项目纯Java编写，无原生二进制依赖，兼容 Windows / Linux / macOS / Android。
 
部分代码由AI辅助开发，GLM 5.2的成本实在太高 :(
